# Video Point Tracker

Программа для отслеживания точки на видео с использованием алгоритма оптического потока Лукаса-Канаде.

## Системные требования

### Минимальные:
- **Процессор:** Intel Core i7-13790F или аналог
- **Видеокарта:** NVIDIA GeForce RTX 2080 Ti или аналог (используется для отображения)
- **ОЗУ:** 8 ГБ
- **ОС:** Windows 10/11, Linux, macOS

### Программные требования:
- **Python:** 3.13 или выше
- **OpenCV:** 4.8.0 или выше
- **NumPy:** 1.24.0 или выше
